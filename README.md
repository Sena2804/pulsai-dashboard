# PulsAI - Intelligent CRM & Marketing Automation

**PulsAI** PulsAI is an intelligent CRM platform that combines conversational AI, ticket management, and marketing automation.

---

## Project Pages

- **Dynamic Dashboard:** Revenue and KPI visualization.

- **Conversational AI:** Chat interface with payment incident monitoring.

- **Ticketing Management:** Inventory tracking with visual progress bars.

- **Marketing Automation:** Automatic sorting of actions (Instagram/WhatsApp/Facebook) by date.

- **Reactive Profile:** Profile update simulation with user feedback.

## Technical Stack
- **Framework:** Next.js 16.1.6 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion (AnimatePresence, smooth transitions)
- **Icons:** Lucide React

---

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.js`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
